Optimizing Search Discoverability via CTR & Engagement Opportunity Scoring

Author: Ayush Narkhede | Track: Machine Learning Internship

1. Abstract

This research evaluates machine learning approaches to identify web pages that under-capture search clicks despite high impressions. Utilizing search intelligence logs from the FlyRank warehouse, we formulated a binary classification task to predict user engagement opportunity scores. A gradient-boosted decision model achieved an ROC-AUC score exceeding 0.78 compared to a rule-based baseline of 0.55. These predictions directly inform metadata optimization and content revision playbooks. All data representations are anonymized and public-safe.

2. Introduction & Problem Statement

Search engine optimization teams frequently struggle to prioritize which high-impression pages require meta-description or title tag updates. Standard rule-based heuristics (e.g., hardcoded Click-Through Rate thresholds) fail to capture non-linear relationships across queries, seasonality, and position decay. This project delivers an automated engagement opportunity scoring engine that flags pages performing below statistical expectations relative to their search ranking position.

3. Data Summary

Data was sourced from the official FlyRank Machine Learning Internship warehouse via Hugging Face DuckDB integration. Aggregations were performed at the document-query impression level. Client identifiers, explicit query strings, and proprietary domain details were scrubbed to ensure total compliance with public safety rules.

4. Methodology

  • Unit of Analysis: Single page-impression interaction session.
  • Target Variable: is_engaged (Binary flag representing target CTR performance above position baseline).
  • Models Trained: Logistic Regression (Baseline) vs. Random Forest & LightGBM Classifiers.
  • Validation: GroupKFold split by session to prevent temporal and domain data leakage.

5. Results

The machine learning approach significantly outperformed fixed baseline heuristics across evaluation metrics:

  • Rule-Based Baseline ROC-AUC: 0.552
  • Logistic Regression ROC-AUC: 0.684
  • LightGBM Classifier ROC-AUC: 0.786

6. Limitations & Honest Framing

The model outputs directional engagement opportunity scores rather than causal impact guarantees. External factors such as competitor content changes, algorithm updates, and UI rendering shifts remain unobserved variables.

7. Ranked Recommendations

  1. Tier 1 (High Score): Immediate title tag and meta-description rewrite.
  2. Tier 2 (Medium Score): Review content freshness and search intent alignment.
  3. Tier 3 (Low Score): Maintain current position and monitor over standard refresh window.
